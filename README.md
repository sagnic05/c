# 💻 C-BREAK OS

> **Learn C Programming by Surviving It.**

Welcome to **C-BREAK OS**, an interactive, gamified curriculum designed to teach C programming fundamentals. This project eliminates the boredom of traditional coding tutorials by turning concepts like loops, pointers, and bitwise operations into real gameplay mechanics. 

Instead of reading textbooks, you will decrypt syntax, fight viruses in turn-based combat, and defend system cores in 3D space.

## 🎮 The Training Modules (Games)

This OS contains three primary training modules, each designed to test different skill levels:

### 1. Syntax Assembler (`assembler.html`)
* **Difficulty:** Beginner
* **Concept:** Muscle Memory & Syntax Structure
* **Gameplay:** Master the basics of C without typing a single line. Decrypt scrambled code blocks by clicking and arranging them into the assembly buffer. 

### 2. Kernel Panic: C-RPG (`rpg.html`)
* **Difficulty:** Intermediate (Tactical Combat)
* **Concept:** Logic, Conditionals, and Loops
* **Gameplay:** A turn-based retro RPG battle system. Face off against the *Syntax Bug* and *Data Titans*. Choose your difficulty level, complete the fragmented code, and execute powerful Logic Strikes to drain the enemy's HP. If you get a Syntax Error, the enemy attacks you.

### 3. Bitwise Assault (`arcade.html`)
* **Difficulty:** Advanced / Pro (Survival Mode)
* **Concept:** Bitwise Operations, Pointers, Memory Allocation
* **Gameplay:** A high-speed, 3D Synthwave arcade shooter built with HTML5 Canvas. The system core is under attack by geometric data-viruses. You must evaluate complex C code in your head and type the exact output to fire your laser before the grid is overrun.

## 🛠️ Tech Stack
This entire suite is built from scratch using pure frontend technologies—no external libraries or game engines were used.
* **HTML5**
* **CSS3** (Custom animations, glassmorphism, glowing neon UI)
* **Vanilla JavaScript** (Game logic, state management, and HTML5 `<canvas>` rendering)

## 🚀 How to Run Locally
Because this project uses vanilla web technologies, running it is incredibly simple:
1. Clone or download this repository to your local machine.
2. Open the folder.
3. Double-click on `index.html` to open the Master OS Hub in your default web browser.
4. Click "Initiate Training" on any of the modules to start playing.

# 💻 C-BREAK OS

> **Learn C Programming by Surviving It.**

Welcome to **C-BREAK OS**, a high-performance, gamified curriculum designed to teach C programming fundamentals through immersive software design. This project weaponizes abstract concepts—like pointers, memory allocation, recursion, and bitwise logic—into tangible survival mechanics. 

Instead of reading static textbooks, you will sort memory payloads under time constraints, de-allocate rogue pointers, compile code to escape security subnets, and defend system cores in a 3D synthwave arena.

---

## 🎮 The Training Modules

C-BREAK OS has evolved into a complete, 9-module operating system suite. Each application is engineered to test and expand a specific layer of your technical clearance:

### 1. Memory Bin Sorter (`sorter.html`) 🆕
* **Clearance:** Introductory
* **Concept:** Data Types & Memory Footprints
* **Gameplay:** A fast-paced, low-stress sorting interface. Categorize incoming variables, literals, and format specifiers by dragging or routing them into their correct memory size bins before the system stack overflows.

### 2. Syntax Match V3.1 (`match.html`)
* **Clearance:** Beginner to Expert (5 Security Layers)
* **Concept:** Cryptographic Deck Pairing
* **Gameplay:** A high-stakes memory matching game. Decrypt the system buffer by pairing C syntax with its exact runtime target. Features 100+ unique pairs, a combo multiplier engine, and auto-shifting neon themes across 5 tiers of difficulty.

### 3. Syntax Assembler (`assembler.html`)
* **Clearance:** Beginner
* **Concept:** Muscle Memory & Structural Logic
* **Gameplay:** Master the building blocks of C without typing. Arrange fragmented code segments into the assembly buffer to compile successful operations.

### 4. Syntax Crush (`crush.html`) 🆕
* **Clearance:** Intermediate
* **Concept:** Byte Allocation Quotas & Defragmentation
* **Gameplay:** A dynamic match-3 logic engine. Align consecutive, valid data blocks to clear byte allocation quotas, sanitize corrupted variables, and defragment system buffers.

### 5. Kernel Panic: C-RPG (`rpg.html`)
* **Clearance:** Intermediate (Tactical Combat)
* **Concept:** Conditionals & Program Flow
* **Gameplay:** A turn-based retro RPG battle system. Face off against *Syntax Bugs* and *Data Titans*. Execute "Logic Strikes" by completing code fragments to drain enemy HP. Beware: a `Syntax Error` triggers a system counter-attack.

### 6. Heap Away (`heap.html`) 🆕
* **Clearance:** Advanced (Hardcore Time-Attack)
* **Concept:** Dynamic Memory Management & Garbage Collection
* **Gameplay:** A high-speed pointer de-allocation survival game. Utilize rapid directional input mechanics to correctly `free()` dangling pointers and prevent critical memory leaks before the buffer collapses.

### 7. Pointer Maze (`maze.html`) 🆕
* **Clearance:** Advanced
* **Concept:** Raw Pointer Arithmetic & Memory Offsets
* **Gameplay:** Navigate a procedurally generated memory grid. Evaluate exact pointer arithmetic and dereference paths to safely route your execution thread to the exit core. Powered by a custom algorithm ensuring absolute procedural solvability.

### 8. Terminal Escape: IDE (`escape.html`) 🆕
* **Clearance:** Elite (Progressive Compiler Room)
* **Concept:** Full-Stack Code Compilation & Security Layering
* **Gameplay:** An intense compilation sandbox spanning 50 progressive security tiers. Write, debug, and execute raw C code directly inside the integrated terminal to satisfy rigorous procedural regex test cases and bypass firewall locks.

### 9. Bitwise Assault V3 (`arcade.html`)
* **Clearance:** Expert (Survival Mode)
* **Concept:** Bitwise Math & Rapid Execution
* **Gameplay:** A high-speed, 3D synthwave arcade shooter built on a custom HTML5 Canvas engine. Evaluate complex bitwise operations in real-time and type the output to neutralize incoming data-viruses.

## 📊 Telemetry, Milestones & Persistence
C-BREAK OS features an advanced, persistent **Telemetry Hub** powered by browser `localStorage`. 
* **Global Metrics:** Tracks absolute compilation times, memory allocation speeds, module completion statuses, and highest survival limits.
* **The Master Tier:** Hitting elite speedrun thresholds (like sub-15-second records in Heap Away and Sorter), reaching clearance Tier 10 in Terminal Escape, or maxing out core modules unlocks the legendary **KERNEL MASTER** status—granting visual flair and evolved interface badges directly inside your OS desktop.

## ⚡ PWA Integration (Offline Survivability)
Engineered for zero downtime, C-BREAK OS operates as a **Progressive Web App (PWA)**. Equipped with a custom service worker and dynamic caching manifest, operators can install the environment locally and execute offline code survival drills without an active network uplink.

## 📡 Operator Uplink
Included in the OS is a live **System Feedback Module** powered by Web3Forms. Operators can transmit field performance data, anomaly reports, and system logs directly to the developer uplink from the main terminal.

## 🛠️ Tech Stack
This entire suite is a **"Pure-Code"** project—built entirely from scratch with **zero external game engines or heavy frameworks**.
* **HTML5** (Canvas rendering engines & structured semantics)
* **CSS3** (Glassmorphism UI overlays, dynamic CRT scanlines, and hardware-accelerated 3D transforms)
* **Vanilla JavaScript** (Asynchronous boot sequences, custom procedural generation engines, regex compilation parsers, and client-side persistence)
* **Native Web Audio API** (Procedurally synthesized retro chimes, error buzzes, and atmospheric synthwave cues)

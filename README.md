# 💻 C-BREAK OS

> **Learn C Programming by Surviving It.**

Welcome to **C-BREAK OS**, a high-performance, gamified curriculum designed to teach C programming fundamentals through immersive software design. This project weaponizes abstract concepts—like pointers, recursion, and bitwise logic—into tangible survival mechanics. 

Instead of reading textbooks, you will decrypt security layers, neutralize viruses in tactical combat, and defend system cores in a 3D synthwave arena.

## 🎮 The Training Modules

C-BREAK OS features four primary training modules, each designed to test a specific layer of your technical clearance:

### 1. Syntax Match V3.1 (`match.html`) 🆕
* **Clearance:** Beginner to Expert (5 Security Layers)
* **Concept:** Cryptographic Deck Pairing
* **Gameplay:** A high-stakes memory matching game. Decrypt the system buffer by pairing C syntax with its exact runtime target. Features 100+ unique pairs, a combo multiplier engine, and auto-shifting neon themes across 5 tiers of difficulty.

### 2. Syntax Assembler (`assembler.html`)
* **Clearance:** Beginner
* **Concept:** Muscle Memory & Structural Logic
* **Gameplay:** Master the building blocks of C without typing. Arrange fragmented code segments into the assembly buffer to compile successful operations.

### 3. Kernel Panic: C-RPG (`rpg.html`)
* **Clearance:** Intermediate (Tactical Combat)
* **Concept:** Conditionals & Program Flow
* **Gameplay:** A turn-based retro RPG battle system. Face off against *Syntax Bugs* and *Data Titans*. Execute "Logic Strikes" by completing code fragments to drain enemy HP. Beware: a `Syntax Error` triggers a system counter-attack.

### 4. Bitwise Assault V3 (`arcade.html`)
* **Clearance:** Advanced / Pro (Survival Mode)
* **Concept:** Bitwise Math & Rapid Execution
* **Gameplay:** A high-speed, 3D synthwave arcade shooter built on a custom HTML5 Canvas engine. Evaluate complex bitwise operations in real-time and type the output to neutralize incoming data-viruses.

## 📊 System Telemetry & Persistence
C-BREAK OS features a persistent **User Telemetry** dashboard. Your highest scores in the Arcade and your Decryption clearance in Syntax Match are saved to your local browser storage. Clearing modules unlocks unique achievement badges that glow within your OS Hub.

## 📡 Operator Uplink
Included in the OS is a live **System Feedback Module** powered by Web3Forms. Operators can transmit field performance data and system logs directly to the developer uplink from the main terminal.

## 🛠️ Tech Stack
This entire suite is a "Pure-Code" project—built from scratch using raw frontend technologies with **zero external game engines**.
* **HTML5**
* **CSS3** (Glassmorphism, CRT scanlines, and CSS 3D transforms)
* **Vanilla JavaScript** (Asynchronous boot sequences, game state management, and Canvas rendering)
* **Native Web Audio API** (Procedurally generated synthwave chimes and feedback cues)
